# Sand Tetris 
这是一个复刻《Sand Blast》（流沙俄罗斯方）玩法的游戏Demo，项目基于`TypeScript`和`Vite`构建。

## 如何运行

1.  安装依赖:
    ```bash
    npm install
    ```
2.  启动开发服务器:
    ```bash
    npm run dev
    ```
3.  在浏览器中打开 `http://localhost:5173`

## 构建

要构建项目，请运行：

```bash
npm run build
```
